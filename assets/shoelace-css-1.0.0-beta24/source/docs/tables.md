### Striped Tables

Use the `table-striped` modifier to add stripes to alternating rows.

```html
<table class="table-striped">
  ...
</table>
```

<table class="table-striped">
  <thead>
    <tr><th>Item</th><th>Price</th></tr>
  </thead>
  <tbody>
    <tr><td>Shoe Freshener</td><td>$4.79</td></tr>

### Bordered Tables
